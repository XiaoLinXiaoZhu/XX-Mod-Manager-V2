# XX Mod Manager - UI 设计文档

> 技术栈：React + TypeScript + Tauri 2
> AI 辅助工具：Agentation + react-scan

## 一、设计哲学

### 核心隐喻：实体卡片收藏

整个应用模拟一个**实体卡片收藏体系**的体验：

- **Mod = 卡片**：哑光质感、有厚度、可翻转，像实体收藏卡
- **Pack = 卡包**：一叠卡片用橡皮筋/卡套绑在一起，可以展开看内容
- **Preset = 收纳盒**：一个盒子装了若干卡包和散卡，盒盖有标签

用户的操作就是在**桌面上摆弄这些卡片**——翻看、分组、装盒、切换。

### 设计原则

1. **轻触即改**：编辑通过悬浮窗完成，不打断浏览，不跳转页面
2. **拟物质感**：卡片有哑光材质、微妙阴影、厚度感，hover 时倾斜响应鼠标位置
3. **空间连续性**：页面切换用顶部滑动条（SectionSelector），内容水平滑移
4. **筛选驱动**：左侧树形目录 + 标签筛选决定显示哪些卡片，不是"进入某个页面"
5. **多窗口自由**：关键场景可弹出独立窗口，像是把卡片从桌面拿到手上仔细看

### 2.2 卡片拟物化交互（新增）

#### 单张卡片（Mod Card）

```
┌─────────────────────────┐
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │ ← 封面图（圆角裁切）
│ ▓▓▓▓▓ 哑光材质 ▓▓▓▓▓▓▓ │
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │
├─────────────────────────┤
│  角色名                  │ ← 底部信息条
│  Mod 名称          [●]  │ ← 启用状态灯
└─────────────────────────┘
  ▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔▔   ← 底部厚度阴影
```

**交互层次：**
- **静止**：哑光质感，微妙内阴影模拟厚度，右下角投射柔和阴影
- **Hover**：卡片根据鼠标位置做 3D 倾斜（perspective transform），表面出现微妙光泽扫过效果（模拟哑光卡在光下转动），轻微抬起
- **点击**：翻转到背面 → 背面显示简要信息 + 启用/禁用切换（继承旧版 FlippableElement）
- **右键**：弹出悬浮操作面板
- **长按/拖拽**：卡片"拿起"效果（放大 + 深阴影 + 跟随鼠标物理旋转）

#### 卡片背面

```
┌─────────────────────────┐
│                          │
│  角色：艾莲              │
│  标签：[泳装] [v1.2]    │
│  快捷键：F1 切换模式     │
│                          │
│  ⚠️ 与"艾莲v2"冲突       │
│                          │
│  ━━━━━━━━━━━━━━━━━━━━━  │
│  [启用 ●]    [详情 →]   │
│                          │
└─────────────────────────┘
```

### 2.3 Pack：卡叠 / 卡包

Pack 在视觉上表现为"一叠卡片"：

```
   ┌─────────────────┐
  ┌─────────────────┐│  ← 可见第二张卡的边缘
 ┌─────────────────┐││
 │ ▓▓ 封面：首张 ▓▓│││  ← 最上面那张显示封面
 │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│││
 ├─────────────────┤││
 │ 📦 艾莲日常合集  │││  ← Pack 名称
 │ 3 张卡片    [▼] ││┘  ← 点击展开
 └─────────────────┘│┘
  └─────────────────┘
```

**交互：**
- **折叠态**：显示为卡叠，只看得到最上面一张 + 后面卡片的边缘
- **Hover**：后面的卡片微微散开（扇形偏移），暗示内容
- **点击展开**：卡片散开成扇形或平铺，可以看到所有卡片
- **拖拽卡片到卡叠上**：动画合入，添加到 Pack
- **从卡叠中拖出卡片**：动画分离，从 Pack 移除

### 2.4 Preset：收纳盒

Preset 在视觉上是一个"盒子"：

```
  ┌───────────────────────────────┐
  │  🏷️ 日常装扮                   │ ← 盒盖标签
  │  ┌──────┐ ┌──────┐           │
  │  │卡叠1 │ │卡叠2 │  散卡...  │ ← 盒子里的内容
  │  │艾莲包│ │简包  │           │
  │  └──────┘ └──────┘           │
  └───────────────────────────────┘
```

**交互：**
- **关闭态**：只看到盒盖标签和盒子外观
- **打开**：盒盖滑开动画，显示内部的 Pack（卡叠）和散装 Mod（散卡）
- **激活 Preset**：盒子发光 + 盒盖标签高亮 → 盒内所有卡片批量"启用"
- **切换 Preset**：旧盒子关闭滑走，新盒子滑入打开

### Section 划分（顶部滑动条）

顶部 SectionSelector 控制三个主要 Section，内容水平滑移切换：

| Section | 内容 | 说明 |
|---------|------|------|
| **仓库** | 游戏选择、Preset 收纳盒视图 | 宏观管理：选游戏、切预设 |
| **Mod 管理** | 卡片网格 + 筛选 | 微观操作：浏览、编辑、启禁 |
| **设置** | 路径配置、存储管理、外观 | 系统配置 |

### 左侧筛选面板

继承旧版 `LeftIndex` 的树形目录设计，筛选逻辑驱动主内容区显示：

```
筛选条件变化
    ↓
过滤 mod 列表
    ↓
卡片网格动态增减（带动画）
```

- 点击"艾莲" → 只显示艾莲的 mod 卡片
- 点击"皮肤" → 只显示皮肤类 mod
- 添加标签"泳装" → 进一步缩小范围
- 多条件取交集

## 五、多窗口设计

### 设计理念

Tauri 原生支持多窗口。核心思路：**主窗口是桌面，子窗口是拿到手上的卡片**。

### 多窗口场景

| 场景 | 触发方式 | 窗口特征 | 用途 |
|------|---------|---------|------|
| **Mod 详情窗** | 右键→"在新窗口打开" / 双击卡片 | 小窗口，可调大小 | 专注查看和编辑单个 mod 的所有信息 |
| **对比窗口** | 多选两个mod→"对比" | 左右分栏窗口 | 对比两个冲突 mod 的差异，辅助选择 |
| **快捷键总览窗** | 从设置或状态栏打开 | 独立窗口，总是在最前 | 游戏中切回来快速查看快捷键 |
| **导入向导窗** | 拖拽文件到主窗口 | 模态子窗口 | 导入新 mod 的流程 |
| **迷你控制器** | 状态栏→"迷你模式" | 小型悬浮窗，置顶 | 游戏中快速切换 preset 和 mod |

### 5.1 Mod 详情窗（独立窗口版）

当用户想仔细查看一个 mod 时，可以弹出独立窗口：

### 5.2 迷你控制器（游戏中快速操作）

这是多窗口带来的最大"小巧思"——**游戏中的浮动控制面板**：

```
┌── XX Mod Manager ── 📌 ─┐
│                           │
│  ★ 当前: 日常装扮         │
│  ┌───────┐ ┌───────┐     │
│  │ 约战  │ │ 全裸  │     │
│  └───────┘ └───────┘     │
│                           │
│  最近切换:                │
│  [●] EllenSlim            │
│  [○] GraceCasual          │
│                           │
│  ⌨️ F1:切换模式 F2:发型   │
└───────────────────────────┘
```

特征：
- **小窗口**，始终置顶（always on top）
- 用户可以在打游戏时 Alt+Tab 快速看到
- 一键切换 Preset 无需打开完整主窗口
- 显示当前 mod 的快捷键提示（减少记忆负担）
- 支持快速启用/禁用最近使用的 mod

```
stores/
```

## 九、技术选型

## 十、视觉风格

- **卡片材质**：哑光质感，通过 CSS 实现 —— 微妙内阴影 + 线性渐变模拟表面纹理 + 底部加深模拟厚度
- **光泽效果**：hover 时一道光泽从左上扫到右下（radial-gradient 跟随鼠标），模拟哑光卡片在光线下转动
- **色调**：深色主题，游戏主题色作为强调色
- **字体**：鸿蒙字体（HarmonyOS Sans）/思源黑体 CJK，标题加粗
- **阴影层次**：静止 → hover → 拖拽 三档阴影深度，模拟物理高度
- **毛玻璃**：悬浮窗和对话框使用 backdrop-filter blur
