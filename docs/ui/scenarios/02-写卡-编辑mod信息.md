# 场景二：写卡 —— 编辑 Mod 信息

## 场景描述

用户想修改一个已有 mod 的名称、标签、快捷键或封面。

## 用户故事

> 我之前导入的 mod 名字太长了，而且没有封面图。我想给它改个好听的名字、配张好看的图、贴几个标签。就像收藏卡玩家在卡套里加自己的注释卡一样。

## 关键设计理念

**编辑 = 翻到卡片背面写字**

不需要打开新页面、弹出模态对话框、离开当前视图。就在原地，翻转那张卡片，在背面写字。

## 交互流程

### 入口：翻卡

在 Mod 管理的卡片网格中，用户看到一张卡片想编辑。

**方式A（触觉版）**：直接在卡片上向右滑动手势（鼠标按住右划），卡片像被手指翻过去一样，翻转到背面。

**方式B（快捷版）**：右键卡片 → 菜单中"写卡"，卡片翻转。

**方式C（点击版）**：单击卡片，卡片翻转（继承旧版 ToggleableCardElement 行为）。

翻转动画：卡片以中间为轴做 3D Y 轴旋转，过程中可以看到卡片的"厚度"（侧边有纸张纹理）。

### 背面：就地编辑

卡片背面就是编辑界面——但它仍然是一张"卡片"的样子，不是一个表单弹窗。

```
  ┌──────────────────────────┐
  │ ┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈  │
  │  EllenSwim_v1            │ ← 点击即编辑，inline 输入框
  │ ┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈┈  │
  │                           │
  │  角色: 艾莲               │ ← 点击弹出选择气泡
  │  类型: 皮肤               │
  │                           │
  │  〔泳装〕〔v1〕〔+添加〕  │ ← 标签贴纸
  │                           │
  │  ⌨ F1 切换    [改]       │ ← 快捷键，点"改"弹出键位选择
  │                           │
  │  ⚠ 与 EllenV2 冲突       │ ← 冲突提示（只读信息）
  │                           │
  │  📁 打开  🗑 删除         │ ← 底部操作
  │                           │
  │  ● 已启用                 │ ← 状态开关
  └──────────────────────────┘
```

**每个字段的编辑方式都像在卡片上"写字"：**

- **名称**：点击文字区域，文字变成可编辑状态，带一条手写风格的下划线。打字时字符逐个出现，有轻微的打字机节奏感。
- **标签**：点击"+添加"，从卡片底部滑上来一组推荐标签（基于角色识别和已有标签自动补全），点选标签 → 标签像贴纸一样"啪"地贴到卡片上。也可以手打。
- **快捷键**：点击"改"，弹出一个小气泡——"按下新按键"，用户按键即完成修改，气泡消失。
- **封面图**：卡片背面没有封面图设置——翻回正面后，在正面长按封面区域，弹出一个"换封面"的小窗（选择图片/截屏/从文件选择）。

### 翻回正面

编辑完成后：
- 点击卡片外任意区域，卡片翻回正面
- 或者按 Esc
- 或者再次滑动/点击

翻回正面的瞬间，正面的内容已经更新了——名称变了、标签变了。如果修改了封面，封面有一个淡入更新效果。

不需要"保存"按钮。每个字段的修改都是即时保存的（自动防抖写入）。

### 换封面（正面操作）

封面是卡片最醒目的部分，值得单独设计：

```
长按封面图区域 (0.5s)
    ↓
封面区域出现毛玻璃叠层 + 操作按钮：

  ┌──────────────────────────┐
  │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │
  │ ▓▓▓░░░░░░░░░░░░░░░░▓▓▓ │
  │ ▓▓░  📁从文件选择   ░▓▓ │
  │ ▓▓░  📋从剪贴板粘贴 ░▓▓ │
  │ ▓▓░  🔄自动检测     ░▓▓ │
  │ ▓▓▓░░░░░░░░░░░░░░░░▓▓▓ │
  │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │
  ├──────────────────────────┤
  │  名称...                  │
  └──────────────────────────┘
```

选择图片后，新图片从选择方向"滑入"覆盖旧封面。

## 多卡批量写卡

选中多张卡片（Ctrl+点击），右键 → "批量写卡"。

多张卡片同时翻转到背面，重叠在一起只露出可编辑的公共字段：

```
  ┌──────────────────────────┐
  │ ┈ 已选中 4 张卡片 ┈┈┈┈  │
  │                           │
  │  角色: [统一修改为...▾]   │
  │                           │
  │  添加标签: 〔+添加〕      │ ← 给所有卡片贴相同标签
  │                           │
  │  状态: [全部启用]         │
  │        [全部禁用]         │
  │                           │
  └──────────────────────────┘
```

## 符合直觉且有趣

- "翻到背面写字"是收藏卡/明信片的通用直觉
- 不离开当前视图——写完翻回来，旁边的卡片都还在
- 标签像贴纸，名字像手写，封面像换照片——每个操作都有物理隐喻
- 没有"保存"按钮——就像在纸上写字，写上去就是写上去了

## 高效

- 翻卡即编辑，零导航成本
- inline 编辑，不弹窗、不跳转
- 自动保存，不需要确认
- 批量操作覆盖多卡场景
- 快捷键修改只需"按一下新键"
