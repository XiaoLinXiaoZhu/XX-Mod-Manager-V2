# 场景五：换装 —— 游戏中切换

## 场景描述

用户正在游戏中，想切换 Preset 或者快速启用/禁用某个 mod，不想退出游戏。

## 用户故事

> 我在打绝区零，突然想换个角色皮肤拍照。我不想关游戏去操作主程序。我想按个快捷键，弹出一个小浮窗，快速切一下预设就好了。如果能顺便看一眼我当前mod的快捷键是什么就更好了。

## 核心意象：从口袋里掏出一张小卡片

游戏中的控制器不是一个"缩小版的主程序"，而是一张**随身携带的卡片**——一张特殊的"遥控卡"。

## 交互流程

### 唤起

用户按下全局快捷键（如 `Ctrl+Shift+M`）。

屏幕角落（默认右下角）滑入一张卡片——这张卡片比普通 mod 卡片稍大一些，它是一张"遥控卡"。

```
滑入动画：从屏幕右边缘滑入，带卡片翻转效果

                              ╭────────────────────────╮
                              │  XX Mod Manager   📌   │
                              │                        │
                              │  ★ 当前: 日常装扮      │
                              │                        │
                              │  ┌──────┐ ┌──────┐    │
                              │  │约战  │ │夏日  │    │
                              │  │模式  │ │海滩  │    │
                              │  └──────┘ └──────┘    │
                              │                        │
                              │  ── 最近 ──           │
                              │  ● EllenSlim           │
                              │  ● HideHUD             │
                              │  ○ GraceCasual         │
                              │                        │
                              │  ── 快捷键 ──         │
                              │  F1 切换模式           │
                              │  F2 切换发型           │
                              │  F3 显示/隐藏          │
                              │                        │
                              ╰────────────────────────╯
```

这张遥控卡的特征：
- **始终在最顶层**（Tauri always-on-top）
- **半透明背景**，不完全遮挡游戏画面
- **可拖拽移动**到屏幕任意位置
- **📌 按钮**：固定/取消固定（固定后不会自动消失）

### 遥控卡的内容

遥控卡分三个区域：

**1. Preset 切换区**

小卡片形式展示可用的 Preset，当前激活的有发光边框。

点击另一个 Preset 小卡片：
- 小卡片亮起，旧的熄灭
- 后台执行 Preset 切换
- 切换完成后，遥控卡上显示一行提示："✓ 已切换到 约战模式"

```
  ┌──────┐ ┌──────┐ ┌──────┐
  │★日常 │ │ 约战 │ │ 夏日 │   ← 点击切换
  │ glow │ │      │ │      │
  └──────┘ └──────┘ └──────┘
```

**2. 最近 Mod 快速切换**

显示最近操作过的 3-5 个 mod，每个旁边有亮/暗状态灯。

点击状态灯即可切换：
```
  ● EllenSlim      ← 点击 ● → 变成 ○（禁用）
  ● HideHUD        
  ○ GraceCasual    ← 点击 ○ → 变成 ●（启用）
```

一次点击完成，没有确认框。

**3. 快捷键备忘区**

显示当前已启用 mod 的快捷键绑定。这是**只读**的——纯粹是提醒用户"F1 是干什么的"。

```
  F1  切换模式 (EllenSlim)
  F2  切换发型 (EllenSlim)
  F3  显示/隐藏 (HideHUD)
```

当 Preset 切换后，快捷键列表自动更新。

### 消失

- 按快捷键再次切换 → 遥控卡滑出屏幕
- 按 Esc → 遥控卡滑出
- 点击遥控卡以外的区域 → 遥控卡滑出（除非📌已固定）
- 固定模式下，遥控卡持续显示，缩小到最小尺寸

### 最小化态

如果用户固定📌了遥控卡但嫌它太大：

点击遥控卡的空白区域，它**折叠**成一个迷你条：

```
  ┌────────────────────────┐
  │ ★日常  ● 12mod  ⌨ F1-3│  ← 一行信息
  └────────────────────────┘
```

再次点击展开回完整卡片。

### 紧急操作

遥控卡底部有一个隐藏区域（向上拖拽展开）：

```
  ── 紧急操作 ──
  [全部禁用]  [重载]
```

- "全部禁用"：一键禁用所有 mod（游戏出 bug 时的紧急按钮）
- "重载"：重新加载配置

## 场景变体：Alt+Tab 回到主程序

如果用户选择 Alt+Tab 回到主程序而不是用遥控卡：

主程序检测到游戏正在运行，会在顶部显示一个"游戏运行中"的横幅：

```
  ┌─ 🎮 绝区零运行中 ─── [返回游戏] ── [迷你控制器] ──┐
  │                                                       │
  │  （正常的主程序界面）                                  │
  │                                                       │
```

用户在主程序中做的任何修改（启用/禁用 mod、切换 Preset）都会实时生效——不需要重启游戏（3dmigoto 支持热重载 F10）。

## 符合直觉且有趣

- "从口袋里掏出一张遥控卡"——快捷键唤出，滑入动画
- 遥控卡本身也是一张卡片——整个应用的卡片隐喻一以贯之
- Preset 切换用小卡片表示，点亮/熄灭——和主界面一致的交互语言
- 快捷键备忘在手边——不用去翻哪个 mod 绑了什么键
- 折叠成迷你条——游戏中不碍事

## 高效

- 全局快捷键唤出，0.3s 可见
- 切换 Preset：看到 → 点击 → 完成，3 步 <1s
- 开关单个 mod：看到 → 点击灯，2 步 <0.5s
- 查看快捷键：唤出即可见，0 额外操作
- Esc 关闭，不需要鼠标去点 ×
