# 场景四：装盒 —— 组织 Pack 和 Preset

## 场景描述

用户想把几个相关的 mod 组织到一起，打成 Pack（卡包），再把几个 Pack 装进 Preset（收纳盒），方便日后一键切换。

## 用户故事

> 我有艾莲的泳装 mod + 泳装武器 mod + 沙滩场景 mod，我想把它们绑成一个"艾莲夏日包"。然后我还有简的约会 mod 包。我要把这两个包和一个 HideHUD mod 一起装进一个"约会模式"的收纳盒里，以后一键切换。

## 核心隐喻

这是一个**整理桌面**的过程：
- 散卡 → 用夹子绑成一叠 = 创建 Pack
- 几叠卡 + 几张散卡 → 装进盒子 = 创建 Preset
- 打开/关闭盒子 → 一键启用/禁用所有里面的 mod

## 交互流程

### 创建 Pack（把散卡绑成一叠）

**方式A：拖叠法（最直觉）**

在桌面上，用户把一张卡片拖到另一张卡片上方，短暂停留（0.5s）。

两张卡片自动**叠在一起**，形成一个小卡叠：

```
  拖拽中:
  
    ┌────┐              ┌────┐
    │卡A │   ──拖到──→  │卡B │
    └────┘              └────┘
    
  松手后:
  
       ┌────┐
      ┌────┐│    ← 两张叠在一起
      │卡A ││
      │    ││
      │新建包│┘    ← 底部标签变成"新建包"
      └────┘
    
  弹出气泡:
     ╭────────────────╮
     │ 包名称:        │
     │ [艾莲夏日___] │
     │       [确定✓]  │
     ╰────────────────╯
```

命名气泡是一个小纸条——写完名字，纸条折叠成卡叠的**腰封**（一条横着的标签条缠在卡叠中间）。

继续拖第三张卡片到这个卡叠上 → 卡叠变厚，数字更新。

**方式B：框选法（高效）**

在桌面上画一个选区框住多张卡片，右键 → "打包"。

被选中的卡片飞向屏幕中央聚拢，叠成一叠，弹出命名气泡。

**方式C：右键菜单**

右键一张卡片 → "加入包..." → 选择已有 Pack，或"新建包"。

### Pack 的外观与交互

卡叠在桌面上的样子：

```
  折叠态（日常状态）:
  
      ┌────────┐
     ┌────────┐│     ← 后面的卡片露出边缘
    ┌────────┐││
    │▓▓封面▓▓│││     ← 最上面一张显示封面
    │▓▓▓▓▓▓▓▓│││
    ├────────┤││
    │艾莲夏日 │┤│     ← Pack 名称
    │ 3张  ▶ ││┘     ← 数量 + 展开按钮
    └────────┘┘
```

**Hover**：后面的卡片微微散开（向右偏移 5px, 10px），暗示里面有更多内容。

**点击展开**：

```
  展开态:
  
    ┌────────┐  ┌────────┐  ┌────────┐
    │ 卡片1  │  │ 卡片2  │  │ 卡片3  │    ← 扇形散开
    │        │  │        │  │        │
    │  ● 亮  │  │  ● 亮  │  │  ○ 暗  │    ← 可以单独启用/禁用
    └────────┘  └────────┘  └────────┘
         ╲         │         ╱
          ╲        │        ╱
           ╲       │       ╱
      ┌────────────────────────────┐
      │  📦 艾莲夏日   [折叠▲]     │     ← Pack 腰封变成了底座
      └────────────────────────────┘
```

展开时，Pack 里的每张卡片可以单独操作（启用/禁用/编辑/拖出去）。

**从 Pack 中移除**：展开后，把一张卡片拖走。卡片脱离卡叠，剩下的卡片重新聚拢。

### 创建 Preset（把卡叠装进盒子）

**方式A：盒子从桌面下方"升起"**

用户点击桌面上的"新建预设"按钮（或右键桌面 → "新建收纳盒"）。

桌面下方**升起**一个空的收纳盒，盒盖打开着，等待放入内容。

```
  ┌─────────────────────────────────────────────────┐
  │                                                  │
  │    ┌────┐  ┌────┐         ← 桌面上的卡片和卡叠  │
  │    │    │  │卡叠│                                │
  │    └────┘  └────┘                                │
  │                                                  │
  │  ┌────────────────────────────────┐              │
  │  │  🏷️ [输入预设名称...]          │ ← 盒盖打开  │
  │  │  ┌─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐  │              │
  │  │  ╎                          ╎  │              │
  │  │  ╎  拖入卡片或卡叠到这里     ╎  │  ← 空盒子  │
  │  │  ╎                          ╎  │              │
  │  │  └─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘  │              │
  │  └────────────────────────────────┘              │
  │                                                  │
  └──────────────────────────────────────────────────┘
```

用户把卡片/卡叠拖进盒子。放入时有"嗒"的落入效果。

**方式B：框选 → 装盒**

框选多个卡片和卡叠 → 右键 → "装盒"。它们飞进新出现的盒子里。

### Preset 盒子在桌面上的样子

```
  关闭态:
  
  ╔══════════════════════╗
  ║  🏷️ 约会模式         ║    ← 盒盖，显示名称
  ║  5 mod · 2 pack      ║    ← 内容摘要
  ║  ○ 未激活            ║
  ╚══════════════════════╝

  激活态:
  
  ╔══════════════════════╗
  ║  🏷️ 约会模式         ║    ← 盒盖，主题色发光
  ║  5 mod · 2 pack      ║
  ║  ★ 已激活  ✦✦✦       ║    ← 发光效果
  ╚══════════════════════╝
```

**点击盒子 → 打开：**

```
  ╔══════════════════════════════════════════╗
  ║  🏷️ 约会模式           [激活★] [✕关闭]  ║  ← 盒盖滑开
  ╠──────────────────────────────────────────╣
  ║                                          ║
  ║    ┌──────┐  ┌──────┐                   ║
  ║    │ 卡叠 │  │ 卡叠 │    ← Pack 们      ║
  ║    │艾莲夏│  │简约会│                   ║
  ║    │  3张 │  │  2张 │                   ║
  ║    └──────┘  └──────┘                   ║
  ║                                          ║
  ║    ┌────┐                               ║
  ║    │散卡│  ← 不属于任何 Pack 的散装 mod  ║
  ║    │HUD │                               ║
  ║    └────┘                               ║
  ║                                          ║
  ╚══════════════════════════════════════════╝
```

打开后可以：
- 拖入新卡片/卡叠
- 拖出移除
- 点击"激活" → 盒子内所有 mod 被启用，盒子发光
- 点击某个 Pack → 展开 Pack 看内容

### 激活 Preset

点击"激活"按钮，或者双击关闭态的盒子。

发生什么：
1. 之前激活的 Preset 盒子的光熄灭
2. 新 Preset 盒子亮起发光
3. 桌面上，属于这个 Preset 的所有卡片"亮起"，不属于的"熄灭"
4. 后台：PresetManager.resolve() → ModLinker 批量操作符号链接

视觉上是**一波灯光切换**——一批灯灭了，另一批亮了。整个桌面的亮/暗分布改变了。

## 符合直觉且有趣

- 叠卡片 = 创建 Pack——生活中整理卡片就是这么做的
- 装进盒子 = 创建 Preset——收纳盒隐喻人人都懂
- 拖拽是主要交互——不是填表单
- 盒子打开/关闭有物理感的动画
- 激活 Preset = 整个桌面灯光变化，仪式感满满

## 高效

- 拖叠两张卡片即可创建 Pack，无需打开任何菜单
- 框选 → 右键 → 打包/装盒，支持批量操作
- Preset 之间双击切换，后台自动处理符号链接
- 盒子关闭态就显示摘要信息，不需要打开就能看到内容概况
